# 使用可能な関数・キーワード 一覧表

## [Grep] 文字列の行内検索

### Ruby
制御構文: `if, else, while, for, def, return, end`  
入出力: `File.open, gets, puts, print, readline`  
文字列操作: `arr[i]`  
演算子: `+, -, *, /, %, ==, !=, <, >, <=, >=`

演算子: `+, -, *, /, %, ==, !=, <, >, <=, >=`

### JavaScript (Node.js)
